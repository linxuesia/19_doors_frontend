## 构建与发布

```bash
# 构建体验版（不要用 dev:weapp，那个是 watch 开发模式）
cd /Users/shanlisi/sia/19_doors/frontend
npm run build:weapp
```

构建完成后在微信开发者工具中点击「上传」→ 填写版本号 → 设为体验版。
